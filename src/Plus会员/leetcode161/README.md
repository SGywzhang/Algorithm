# LeetCode 161. 相隔为 1 的编辑距离 (One Edit Distance)

## 题目描述

给定两个字符串 s 和 t，判断它们是否恰好只相差一个字符的编辑操作（插入、删除或替换）。

### 输入输出格式
- 输入：两个字符串 s, t
- 输出：布尔值，是否只相差一个编辑操作

### 示例
```
输入: s = "ab", t = "acb"
输出: true
解释: 可以通过在 s 的第一个位置插入 'c' 得到 t。

输入: s = "cab", t = "ad"
输出: false

输入: s = "1203", t = "1213"
输出: true
```
