# LeetCode 302. 包含全部黑色像素的最小矩形 (Smallest Rectangle Enclosing Black Pixels)

## 题目描述

给定一个二维二进制矩阵，其中 '1' 表示黑色像素，'0' 表示白色像素。找到包含所有黑色像素的最小矩形。

### 输入输出格式
- 输入：二维字符数组 image，整数 x, y（任意一个黑色像素的坐标）
- 输出：整数，最小矩形的面积

### 示例
```
输入: image = [["0","0","1","0"],["0","1","1","0"],["0","1","0","0"]], x = 0, y = 2
输出: 6
```
