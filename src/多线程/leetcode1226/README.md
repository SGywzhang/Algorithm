# LeetCode 1226. 哲学家进餐问题 (The Dining Philosophers)

## 题目描述

5 个哲学家围坐在圆桌前，每人面前有一只碗和一只筷子。

每位哲学家只需要两只筷子就能进餐。

设计一个多线程程序，模拟哲学家进餐问题，要求：
- 每位哲学家必须先拿起左边的筷子，再拿起右边的筷子。
- 只有拿到两只筷子才能吃饭。
- 吃完饭后，必须先放下左边的筷子，再放下右边的筷子。
- 保证不会发生死锁。

---

**示例 1：**

```
输入: 无
输出: 每位哲学家都能正常进餐，无死锁
```

**提示：**
- 1 <= 哲学家数量 <= 5
- 每位哲学家最多进餐一次
